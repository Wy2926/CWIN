# CWIN 产品需求文档（PRD）v1.0 草案

> 状态：待评审 ｜ 日期：2026-07-09 ｜ 平台：Windows 11（兼容 21H2 老任务栏）

---

## 1. 产品定位

**CWIN** 是一款 Windows 11 任务栏增强软件，把有限的任务栏区域变成**高信息密度、高度自定义的主要信息载体与快捷操作入口**。UI 兼容性与体验目标对标 macOS 菜单栏 + 灵动岛（药丸），视觉风格遵循 Win11 Fluent 设计语言：大卡片、Acrylic/Mica 毛玻璃、圆角、简约大气。

**核心差异化**：
- 真正"长在"任务栏里（注入式融合），而非悬浮小部件
- 胶囊不是一个小点：支持迷你图表、股票、天气、硬件信息等丰富展示
- 胶囊可轮换、切换、（未来）跟随前台应用变化
- 未来开放 SDK 插件生态

**硬性约束**：低内存占用（目标常驻 < 30MB）。

## 2. 三大模块（产品全景）

| 模块 | 说明 | 优先级 |
|---|---|---|
| 信息胶囊 | 任务栏常驻状态胶囊（CPU、天气、日程、消息数等） | **MVP** |
| 大卡片面板 | 点击胶囊弹出对应的毛玻璃大卡片（通知/控制中心风格） | **MVP** |
| 增强启动区 | 自定义图标组、文件夹、快捷操作（Dock/启动器） | v2 |

## 3. 已锁定的关键决策

| 决策项 | 结论 |
|---|---|
| 任务栏融合方式 | **A 注入式**（注入 Explorer / Shell_TrayWnd），胶囊真正嵌入任务栏 |
| 安全降级 | **MVP 必备**：注入失败自动切换为伴生窗口模式，用户无感知 |
| 技术栈 | **C++ 原生 + Direct2D/DirectComposition** 渲染，毛玻璃用 DWM/DirectComposition 原生实现 |
| MVP 模块 | 信息胶囊 + 大卡片面板 |
| 大卡片交互深度 | MVP **只读展示为主** |
| 系统版本 | Win11 全版本：**21H2 老任务栏 + 22H2/23H2 新任务栏**双适配层 |
| 设置界面 | **WebView2** 设置页（低频使用，打开时加载、关闭即释放内存） |
| 配置存储 | 本地 **JSON**（便于手改/备份/未来云同步） |
| 在线数据源 | MVP 用免费无 key 源（天气：Open-Meteo） |
| 分发 | **安装包 + 自动更新检查** |
| 产品名 | 沿用 **CWIN** |
| 开发模式 | ECC 开发模式；黑盒测试由产品负责人手动执行 |

## 4. MVP 范围定义

### 4.1 内置胶囊（首发 4 个，验证信息模型）
1. **硬件监控**：CPU / 内存 / GPU 占用，迷你折线图（sparkline）
2. **天气**：图标 + 温度，数据源 Open-Meteo
3. **时钟 / 日程**：自定义格式时间，今日日程摘要
4. **网速**：实时上/下行速率

第二批（v1.x）：股票、消息数/未读通知等。

### 4.2 胶囊渲染模板（4 种）
- 纯文本
- 图标 + 文本
- 迷你图表（sparkline）
- 进度环

每个胶囊 = **数据源 + 渲染模板 + 交互行为** 三元组，这一信息模型也是未来插件 SDK 的基础。

### 4.3 交互
- **左键点击胶囊** → 弹出对应毛玻璃大卡片（详细信息，只读为主）
- **右键胶囊** → 配置菜单
- **拖动** → 胶囊排序
- **定时轮换**：多胶囊按可配置间隔轮换展示（"跟随前台应用"规则引擎放 v2）

### 4.4 大卡片面板
- Win11 通知/控制中心风格：毛玻璃、圆角、大卡片布局
- 每种胶囊对应一张详情卡（如硬件卡展示完整历史曲线、天气卡展示多日预报）
- MVP 只读展示；操作类能力（切换城市、结束进程等）放 v2

### 4.5 系统适配（MVP 边界）
- 支持：主显示器、默认任务栏布局（居中/左对齐两种对齐方式）
- 暂不支持（排期 v1.x）：多显示器任务栏、自动隐藏任务栏、小任务栏尺寸

### 4.6 设置页（WebView2）
- 胶囊启用/禁用与管理、排序、轮换间隔、主题（浅色/深色/跟随系统）
- 开机自启开关、更新检查

### 4.7 可靠性
- 注入失败 / Explorer 重启 → 自动检测并恢复或降级为伴生窗口
- Windows 更新导致注入点变化 → 降级模式保底 + 自动更新推送适配

## 5. 非功能指标

| 指标 | 目标 |
|---|---|
| 常驻内存 | < 30MB（设置页 WebView2 除外，关闭即释放） |
| 常驻 CPU | 空闲 < 0.5%，刷新时瞬时峰值可控 |
| 渲染 | Direct2D/DirectComposition 硬件加速，动画 60fps |
| 启动 | 开机自启，注入完成 < 2s |

## 6. 路线图

- **v1.0（MVP）**：4 内置胶囊 + 4 模板 + 大卡片（只读）+ 定时轮换 + 降级机制 + WebView2 设置页 + 安装包/自动更新
- **v1.x**：多显示器、自动隐藏任务栏适配、股票/消息胶囊、大卡片快捷操作
- **v2**：增强启动区（Dock）、"跟随前台应用"规则引擎、主题市场
- **v3**：**插件 SDK**——进程外插件模型（插件独立进程，崩溃不影响主程序；JSON/IPC 协议，任何语言可写插件），开放数据源/模板/卡片三层扩展点

## 7. 风险

| 风险 | 应对 |
|---|---|
| Win11 更新打破注入 | 降级伴生窗口模式保底 + 自动更新快速跟进 |
| 双代任务栏（21H2/22H2+）维护成本 | 注入层做适配抽象，同时为未来 Win12 变动打底 |
| 免费天气源限流/停服 | 数据源层可插拔，可切换备用源 |
